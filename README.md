## Jira Cloud Migration Simulation

This project includes a Jira Cloud CSV migration simulation report documenting:

- Source CSV preparation
- Jira Cloud CSV import flow
- Field mapping
- Work type/value mapping
- User migration handling
- Imported issue validation
- Backlog, board, list, and issue-detail verification
